To understand your timetable, please note:

L = Lecture
P = Practical
S = Seminar
T = Tutorial
W = Workshop
PB = Problem-based
FW = Field Work
CL = Clinical Laboratory
CP = Clinical Placement
CR = Clinical Practice
CC = Concert Class
IC = Instrument Class
RH = Rehearsal
SC = Screening
ST = Studio
  • The letter indicates the type of activity (see legend above)
  • The first number indicates which activity it is, eg.
    L1 stands for the first weekly lecture for the subject
    L2 stands for the second weekly lecture for the subject
    P1 stands for the first weekly practical session for the subject, etc.
  • The second number (after the slash) indicates the different streams or repeats available for that activity. You only need to attend one stream for each activity, eg.
    L1/01 stands for Lecture 1, Stream 1
    L1/02 stands for Lecture 1, Stream 2, etc.
